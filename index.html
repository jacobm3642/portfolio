VR Escape Room

For my VR escape room project, I focused on backend programming, particularly in designing the interactive elements. I developed an intuitive keypad system that challenges players to solve puzzles and unlock doors within the virtual environment. The backend implementation involves creating a robust code structure for handling user input, validating solutions, and managing game progression.

Beat Saber Clone

In my Beat Saber clone project, I focused on the backend programming that drives the gameplay. One of the notable aspects is the utilization of bitmap images to read in the beat data in binary format. This backend feature enables precise synchronization between the music beats and the gameplay elements. Despite my emphasis on backend programming, the project also showcases my ability to integrate frontend components to create an enjoyable player experience.
